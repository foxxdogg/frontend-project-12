## Описание
Этот проект — упрощенный аналог мессенджера Slack. Позволяет пользователям регистрироваться, создавать публичные каналы и обмениваться сообщениями в реальном времени.

**Основные возможности:**
*   Авторизация и регистрация пользователей (JWT).
*   Создание, удаление и переименование каналов.
*   Обмен сообщениями в реальном времени через WebSockets.
*   Полная локализация интерфейса (Русский/Английский).
*   Валидация форм на стороне клиента.

## Стек технологий
*   **Frontend:** React (Hooks), Redux Toolkit.
*   **Маршрутизация:** React Router.
*   **Формы:** Formik + Yup.
*   **Стили:** Bootstrap + React-Bootstrap.
*   **Связь:** Socket.io-client, Axios.
*   **Интернационализация:** i18next.
